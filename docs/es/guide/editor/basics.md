# El Editor

El editor de LoneWriter ha sido diseñado para minimizar las distracciones y maximizar el flujo creativo, integrando las herramientas de inteligencia artificial de forma orgánica.

---

## Interfaz Principal

### El Área de Escritura
El corazón de la aplicación es un editor de texto enriquecido que admite formatos básicos. Su diseño limpio permite centrarse exclusivamente en la palabra.

### Barra de Herramientas Inteligente
Ubicada en la parte superior o lateral, proporciona acceso rápido a las funciones clave:
- **[Oráculo](../analysis/oracle.md)**: Para consultas de lore.
- **[MPC](../worldbuilding/mpc.md)**: Monitor de propuestas para el [Compendio](../worldbuilding/compendium.md).
- **[Reescritura](./rewrites.md)**: Herramientas para pulir el texto seleccionado.
- **[Ajustes de IA](../setup/api-keys.md)**: Acceso rápido para cambiar de modelo.

---

## Gestión de Escenas y Capítulos
LoneWriter organiza el trabajo en unidades lógicas para facilitar la navegación y la indexación de datos.

- **Navegador**: Permite saltar entre escenas rápidamente.
- **Sinopsis**: Cada escena puede tener un resumen breve que ayuda al Oráculo a entender la progresión de la trama sin necesidad de leer todo el texto cada vez.

---

## Guardado Automático y Local
La seguridad de sus textos es prioritaria.
- **Persistencia**: Los datos se guardan automáticamente en la base de datos local de su navegador (IndexedDB).
- **Estado de Guardado**: Un indicador visual le confirmará que su trabajo está seguro tras cada cambio.

> [!NOTE] NOTA
> Aunque el guardado es automático, se recomienda realizar exportaciones periódicas o configurar la sincronización con Google Drive para mayor seguridad.
