# The Compendium

The Compendium constitutes the knowledge base of your world. It is an intelligent database that LoneWriter uses to maintain narrative consistency and provide accurate information to the Oracle.

---

## Structure of the Compendium
The Compendium organizes information into four main categories:

1. **Characters**: Protagonists, antagonists, and supporting characters.
2. **Locations**: Geographical environments, cities, or specific rooms.
3. **Objects**: Items, technology, or relics relevant to the plot.
4. **Lore**: Historical facts, magic systems, or cultural concepts.

---

## Intelligent Auto-complete
This feature allows artificial intelligence to assist in creating Compendium entries based on the current draft.

### Usage Instructions:
1. Open an entry in the Compendium.
2. Verify that there is related text in the draft.
3. Select the "AI Auto-complete" option.
4. The system will analyze the draft and extract relevant information to complete the fields automatically.

> [!TIP]
> Automatically completed fields can be edited later to ensure total alignment with the author's vision.

---

## Relationships between Entities
LoneWriter allows defining links between different Compendium elements to build a solid narrative network.

### Create a Relationship:
1. Access the Relationships section within an entity.
2. Select the target entity.
3. Define the type of link (example: "Ally of," "Located in").

These relationships are visualized graphically in the Nexus, allowing the structure of the story to be analyzed visually.
